# AI Tools - Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Start Backend
```bash
cd backend
npm start
```
Backend runs on `http://localhost:5000`

### Step 2: Start Frontend
```bash
npm run dev
```
Frontend runs on `http://localhost:5173`

### Step 3: Use AI Tools
1. Login to the app
2. Click "AI Tools" in sidebar
3. Click any tool card
4. Tool opens in new page - start using!

## 🎯 Try These First

### 1. Study Assistant (Easiest)
- Click "Study Assistant" card
- Type: "Explain photosynthesis in simple terms"
- Get instant AI response

### 2. Quiz Generator (Most Fun)
- Click "Quiz Generator" card
- Enter topic: "World War II"
- Set 5 questions, medium difficulty
- Click "Generate Quiz"
- Take the quiz and see your score!

### 3. Flashcards (Most Interactive)
- Click "Flashcard Generator" card
- Enter topic: "Spanish Basic Verbs"
- Set 10 cards
- Click "Generate Flashcards"
- Click cards to flip them!

### 4. Notes Summarizer (Most Useful)
- Click "Notes Summarizer" card
- Paste your long notes OR upload PDF
- Click "Summarize Notes"
- Get concise summary with key points

## 🔧 Troubleshooting

### Tool not loading?
- Check backend is running on port 5000
- Check frontend is running on port 5173
- Verify you're logged in

### "Failed to generate" error?
- Check Gemini API key in `backend/.env`
- Wait a minute (rate limit)
- Try again

### PDF upload not working?
- Ensure file is valid PDF
- Check file size < 10MB
- Try different PDF
